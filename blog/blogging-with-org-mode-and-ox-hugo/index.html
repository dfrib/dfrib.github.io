A blog is born

emacs
Tags// emacs, org, hugo,

I few weeks ago I decided to try to set up a personal blog, aimed primarily at software development topics, but perhaps also containing one or another post about my other interests; climbing/bouldering and writing and performing music.

I intend for the blog to be a place that can encourage me to write up ideas and experiences as they pass through in my day-to-day life, however without my usual need to thoroughly explain every essential background detail for the potential reader. Thus, the posts herein will mostly take the form of shorter insights intended as future reference for myself as well as, hopefully, for some potential blog readers. For more technical topics, I will refer to external links for a deeper dive into the details beyond what the content of the particular blog post covers.

Now, as I’m an avid Emacs user it comes as a natural choice to write this blog using org-mode, particularly by means of literate programming via org-babel. The site is generated using Hugo, a lean open-source static site generator, in combination with ox-hugo as an Org exporter backend to Hugo-compatible Markdown. As for the blog theme, a slightly tweaked version of the (no longer maintained) hugo-redlounge Hugo theme by Tom Maiaroto is used.

Finally, returning to the intended purpose of the blog, I think I can likewise describe it as trying to capture some pieces from my stream of consciousness. As I find bits of awareness to be a good attempt describing one’s consciousness, I think this makes for a good blog title.

Let’s hope for long-living and at least somewhat frequently updated journey together, dear Blog!
